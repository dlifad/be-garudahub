ALUR SINGKAT ADMIN (PLAYER -> TOURNAMENT -> MATCH -> LINEUP)
1. Siapkan data master pemain (players).
2. Assign pemain ke turnamen (tournament_players) dan isi jersey number.
3. Buat match untuk turnamen terkait (matches).
4. Saat set lineup match, ambil pemain berdasarkan tournament_id match, bukan all players.
5. Simpan pemain ke match_lineups via endpoint lineup.
6. Detail match menampilkan lineup sesuai data match tersebut.

CATATAN PENTING
- players.json sekarang format ideal: data master pemain saja.
- relasi ke turnamen dipisah di tournament_players.json.
- backend memvalidasi pemain harus terdaftar di squad turnamen sebelum bisa masuk lineup match.
